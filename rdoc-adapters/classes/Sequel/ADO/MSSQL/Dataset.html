Use a nasty hack of multiple SQL statements in the same call and having the last one return the most recently inserted id. This is necessary as ADOâ€™s default :provider uses a separate native connection for each query.

def insert(*values)
  return super if @opts[:sql]
  with_sql("SET NOCOUNT ON; #{insert_sql(*values)}; SELECT CAST(SCOPE_IDENTITY() AS INTEGER)").single_value
end
