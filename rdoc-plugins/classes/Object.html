The #sql_expr extension adds the #sql_expr method to every object, which returns an wrapped object that works nicely with Sequelâ€™s DSL by calling Sequel.expr:

1.sql_expr < :a     # 1 < a
false.sql_expr & :a # FALSE AND a
true.sql_expr | :a  # TRUE OR a
~nil.sql_expr       # NOT NULL
"a".sql_expr + "b"  # 'a' || 'b'

To load the extension:

Sequel.extension :sql_expr
